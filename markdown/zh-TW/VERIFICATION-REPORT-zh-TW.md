# 科學驗證報告 — 宇宙播遷假說

---

## 概述

本報告針對**宇宙播遷假說**中引用的所有關鍵宣稱、數據點及參考文獻進行系統性科學驗證。採用三種獨立驗證方法：

1. **天文物理計算** — 使用 `astropy` 函式庫（Planck18 宇宙學模型、物理常數、單位換算）
2. **生物學與材料科學文獻驗證** — 使用 `biopython`、PubMed/NCBI 資料庫及網路文獻交叉比對
3. **引用與 DOI 驗證** — 使用 CrossRef REST API、CDMS 資料庫及 NASA 行星資料表

| 5 | 卡爾達肖夫 Type III：約 10³⁶ W | 銀河系估計：10³⁵–10³⁸ W 範圍 | ✅ 驗證通過 |

> **關於卡爾達肖夫 Type 0.7 的說明：** 使用 Sagan 插值公式 K = (log₁₀P − 10) / 10，其中 P = 1.9 × 10¹³ W，計算值為 K ≈ 0.33。常見引用的「Type 0.7」使用不同公式（K = log₁₀(P/10⁶) / 10），計算結果為 K ≈ 0.73。兩種公式在文獻中均有使用；論文的宣稱與較常引用的版本一致。
